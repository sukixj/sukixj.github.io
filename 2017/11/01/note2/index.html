NOI Linux 评测注意事项

发表于 2017-11-01 | 阅读次数:

  • 文件输入输出

  • long long -> %lld ( %I64d会被自动识别成%d )

  • unsigned long long -> %llu

  • 头文件要写全

  • 文件名及后缀必须使用小写

  • 代码限制100KB（打表）

  • 使用的函数和变量名不要和Linux中的内置函数和变量名冲突，如pipe、fork等名称

  • 不要使用#include 头文件，会引入许多可能与自己命名产生冲突的函数和变量。

  • Linux系统下rand()函数与Windows不同 -> (rand()<<15)^rand()
